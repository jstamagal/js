Read the current in-process task list.

Use this before updating todos when you need to avoid duplicates or understand
current progress.

Use for:
- Checking pending, in-progress, and completed work.
- Resuming a multi-step task after interruption.
- Answering operator questions about progress.
- Confirming what remains before marking new items complete.

Output:
- Returns all current todos with content and status.
- Returns `No todos.` when no task list exists.
- The list is process state, not a project file.
